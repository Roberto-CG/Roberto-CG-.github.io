Utiliza nuestra página web para gestionar comodamente las Áreas comerciales

(Aplicación CRUD)



Utilidades:

Sólo necesitas registrarte mediante tu correo electrónico y tendrás acceso a introducción, modificación y borrado de los datos que consideres oportuno, de una manera sencilla.
Puedes codificar e identificar los territorios y editarlos. También puedes crear nuevos y borrar ya existentes.

Entra en la página y la primera vez que lo hagas registrate con un correo electrónico y una clave. De inmediato se enviará un mensaje a esa dirección de confirmación que serviará también de enlace. De esta forma, cada vez que te conectes con la página, introduciendo tu correo y contraseña, ya estarás capacitado para utilizar las herramientas disponibles.


Tecnología utilizada:

Esta página web está en un respositorio de GitHub (como esta propia página que estás leyendo): https://github.com .
La página está publicada en fibase (https://firebase.google.com) y se utiliza una base de datos de su sistema de base de datos (firestore) donde se guardan todas las modificaciones.
